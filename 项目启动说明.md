# 项目启动说明

## 项目简介

这是一个使用 Python 和 Pygame 库开发的贪吃蛇游戏。玩家通过控制蛇的移动来吃食物，增加分数并避免碰撞。

## 环境要求

- Python 3.x
- Pygame 库

## 安装步骤

1. 确保已安装 Python 3.x。
2. 安装 Pygame 库：
   ```bash
   pip install pygame
   ```

## 运行项目

在终端中导航到项目目录并运行以下命令：

```bash
python snake_game.py
```

## 控制说明

- 使用方向键控制蛇的移动。
- 按 `ESC` 键暂停游戏。
- 按 `Q` 键退出游戏。

## 注意事项

- 确保在运行游戏时，窗口处于活动状态。
